TaskAct4
========

TaskAct4 はタスク管理ツールです。

日々の業務の生産性を高めるツールです。

## Description
TaskChute2 にインスパイアされて開発が始まりました。

## Requirement
* Windows 10
* Visual Studio 2015

## Usage
* タスクを登録できます
* タスクを削除できます
* 繰り返しタスクを登録できます
